LESSON 01
What is Python?
The world's most beginner-friendly language
Created in 1991
Python was created by Guido van Rossum. It's named after the comedy show Monty Python's Flying Circus — not the snake!
Why Python?
Python reads almost like English. You don't need to write complex code just to say Hello to the world. That's why beginners love it.
Where is Python used?
Google, NASA, Instagram, Netflix, YouTube — all use Python. It powers Artificial Intelligence, websites, games, data science, and automation.
Running Python

LESSON 02
Variables & Data Types
Storing information in your program
What is a Variable?
A variable is like a labelled box that stores data. You give it a name, and put something inside it using =.
int — Integers
age = 15 — Whole numbers (no decimal). Used for counting, scores, years.
float — Decimal Numbers
height = 5.7 — Numbers with a decimal point. Used for measurements, percentages.
str — Strings (Text)
name = "Priya" — Any text inside quotes. Used for names, messages, addresses.
bool — True or False
passed = True — Only two values: True or False. Used for yes/no decisions.
Check the type
Use type(variable) to find what kind of data is stored inside it.
📖 Example Code
# Variables and Data Types
name = "Alex"
age = 15
marks = 92.5
passed = True

print("Name:", name)
print("Age:", age)
print("Marks:", marks)
print("Passed:", passed)
print("Type of age:", type(age))
print("Type of marks:", type(marks))
📤 Expected Output
Name: Alex Age: 15 Marks: 92.5 Passed: True Type of age: <class 'int'> Type of marks: <class 'float'>

LESSON 03
Operators & Expressions
Making Python do the maths for you
Arithmetic Operators
+ add   - subtract   * multiply   / divide   ** power   % remainder
Comparison Operators
== equal   != not equal   > greater   < less   >= greater-equal   <= less-equal
Logical Operators
and — both must be True  |  or — at least one True  |  not — flips True/False
Order of Operations
Python follows BODMAS: Brackets → Powers → Multiply/Divide → Add/Subtract. Use () to control order.
The % (Modulus)
10 % 3 gives 1 because 10 ÷ 3 = 3 remainder 1. Very useful for checking even/odd numbers!
📖 Example Code
# Operators in action
a = 20
b = 6

print("Add:", a + b)
print("Subtract:", a - b)
print("Multiply:", a * b)
print("Divide:", a / b)
print("Power:", a ** 2)
print("Remainder:", a % b)

# Is a greater than b?
print("a > b:", a > b)
# Is a even?
print("a is even:", a % 2 == 0)
📤 Expected Output
Add: 26 Subtract: 14 Multiply: 120 Divide: 3.3333333333333335 Power: 400 Remainder: 2 a > b: True a is even: True

LESSON 04
Input & Output
Talking to your program
The print() function
print() displays anything on the screen. You can print text, numbers, or variables. Separate items with commas.
The input() function
input() pauses the program and waits for the user to type something. It always returns a string.
Converting Input
Since input() returns text, use int() or float() to convert it: age = int(input("Age: "))
f-Strings (Formatted Strings)
The cleanest way to mix text and variables: f"Hello {name}, you are {age} years old!"
end and sep in print
print("Hi", end="") stops the newline. print(1, 2, 3, sep="-") prints 1-2-3.
📖 Example Code
# Input and Output
# (In this runner, input() uses preset values)
name = "Rahul"
birth_year = 2010

current_year = 2025
age = current_year - birth_year

print(f"Hello, {name}!")
print(f"You are {age} years old.")
print(f"By 2030, you will be {2030 - birth_year}.")

# Formatting numbers
marks = 456.789
print(f"Your marks: {marks:.2f}")  # 2 decimal places
📤 Expected Output
Hello, Rahul! You are 15 years old. By 2030, you will be 20. Your marks: 456.79

LESSON 05
If / Elif / Else
Making decisions in your code
What are Conditions?
Conditions let your program choose what to do based on data — just like YOU decide what to wear based on the weather!
if statement
Runs a block of code only if the condition is True. The colon : and indentation (4 spaces) are mandatory.
else statement
Runs when the if condition is False. It's the 'otherwise' part.
elif (else-if)
Checks a new condition when the previous if was False. You can have many elif blocks.
Nested Conditions
You can put an if inside another if. This is called nesting — use it carefully to avoid confusion.
Indentation is KEY!
Python uses 4 spaces of indentation to define code blocks. Unlike other languages, this is not optional — it will cause an error!
📖 Example Code
# Grade Calculator
marks = 78

if marks >= 90:
    grade = "A+"
    remark = "Outstanding!"
elif marks >= 75:
    grade = "A"
    remark = "Excellent!"
elif marks >= 60:
    grade = "B"
    remark = "Good work!"
elif marks >= 45:
    grade = "C"
    remark = "Keep improving!"
elif marks >= 33:
    grade = "D"
    remark = "Just passed."
else:
    grade = "F"
    remark = "Better luck next time."

print(f"Marks: {marks}")
print(f"Grade: {grade}")
print(f"Remark: {remark}")
📤 Expected Output
Marks: 78 Grade: A Remark: Excellent!

LESSON 06
Loops (for & while)
Repeating actions without repeating yourself
Why Loops?
Imagine printing numbers 1 to 100. Without loops, you'd write 100 print() lines. With a loop, just 2 lines!
for loop
Repeats for each item in a sequence. range(1, 11) gives numbers 1 to 10. range(n) gives 0 to n-1.
while loop
Keeps repeating while a condition is True. Use when you don't know beforehand how many times to repeat.
break
break immediately exits the loop. Useful for stopping when a condition is met.
continue
continue skips the rest of the current iteration and goes to the next one.
Nested Loops
A loop inside a loop. Used for patterns, tables, and grids. The inner loop completes fully for each outer iteration.
📖 Example Code
# Loops demo
# 1. Multiplication Table
n = 7
print(f"--- Table of {n} ---")
for i in range(1, 11):
    print(f"{n} x {i:2} = {n*i:3}")

# 2. Sum of first 10 numbers
total = 0
for i in range(1, 11):
    total += i
print(f"\nSum of 1 to 10 = {total}")

# 3. Pattern
print("\nStar Pattern:")
for i in range(1, 6):
    print("* " * i)
📤 Expected Output
--- Table of 7 --- 7 x 1 = 7 7 x 2 = 14 7 x 3 = 21 7 x 4 = 28 7 x 5 = 35 7 x 6 = 42 7 x 7 = 49 7 x 8 = 56 7 x 9 = 63 7 x 10 = 70 Sum of 1 to 10 = 55 Star Pattern: * * * * * * * * * * * * * * *

LESSON 07
Strings
Working with text in Python
String Basics
Strings are text enclosed in single 'hello' or double "hello" quotes. Both work the same way.
String Indexing
Each character has a position starting from 0. s[0] is first, s[-1] is last. Python also counts from the end!
String Slicing
s[1:4] gives characters at positions 1, 2, 3 (not 4). s[:3] means from start, s[3:] means till end.
String Methods
.upper(), .lower(), .strip(), .replace(), .split(), .find() — Python has many built-in string tools.
String + len()
len("hello") gives 5. You can also use + to join strings (concatenation) and * to repeat.
📖 Example Code
# Strings in Python
name = "Python Programming"

print("Original:", name)
print("Upper:", name.upper())
print("Lower:", name.lower())
print("Length:", len(name))
print("First char:", name[0])
print("Last char:", name[-1])
print("Slice [0:6]:", name[0:6])
print("Replace:", name.replace("Python", "Awesome"))
print("Split:", name.split(" "))
print("Starts with P:", name.startswith("P"))
📤 Expected Output
Original: Python Programming Upper: PYTHON PROGRAMMING Lower: python programming Length: 18 First char: P Last char: g Slice [0:6]: Python Replace: Awesome Programming Split: ['Python', 'Programming'] Starts with P: True

LESSON 08
Lists
Storing multiple values in one place
What is a List?
A list stores many values in one variable, in order. Like a shopping list! Written with square brackets: [1, 2, 3]
Indexing & Slicing
Same as strings — list[0] is first item, list[-1] is last. Slicing works the same way too.
Adding Items
.append(item) adds to the end. .insert(pos, item) adds at a specific position.
Removing Items
.remove(value) removes by value. .pop(index) removes by position (default: last item).
Useful List Methods
.sort() sorts in place. .reverse() reverses. len(list) gives count. sum(list) adds all numbers.
Looping a List
Use for item in list: to go through each element one by one — very powerful!
📖 Example Code
# Lists in Python
marks = [85, 92, 78, 95, 88]

print("Marks:", marks)
print("Highest:", max(marks))
print("Lowest:", min(marks))
print("Total:", sum(marks))
print("Average:", sum(marks) / len(marks))

# Add and remove
marks.append(91)
print("After append:", marks)
marks.sort()
print("Sorted:", marks)

# Loop through list
subjects = ["Math", "Science", "English"]
for sub in subjects:
    print("Subject:", sub)
📤 Expected Output
Marks: [85, 92, 78, 95, 88] Highest: 95 Lowest: 78 Total: 438 Average: 87.6 After append: [85, 92, 78, 95, 88, 91] Sorted: [78, 85, 88, 91, 92, 95] Subject: Math Subject: Science Subject: English

LESSON 09
Functions
Write once, use anywhere
What is a Function?
A function is a named block of reusable code. Instead of writing the same code 10 times, put it in a function and call it 10 times!
Defining a Function
Use def keyword, then the name, then parentheses and colon. The code inside must be indented.
Parameters & Arguments
Functions can accept inputs called parameters. When you call the function, you pass arguments for those parameters.
The return Statement
return sends a result back to the caller. Without return, the function returns None.
Default Parameters
You can give parameters a default value: def greet(name="Student") — works even if no argument is passed.
Why Functions?
They make code reusable, organised, and easier to test and fix. Large programs are built from many small functions.
📖 Example Code
# Functions in Python
def greet(name, age=15):
    return f"Hello {name}! You are {age} years old."

def calculate_area(length, width):
    area = length * width
    return area

def is_even(number):
    if number % 2 == 0:
        return True
    else:
        return False

# Calling functions
print(greet("Ravi"))
print(greet("Meera", 16))

area = calculate_area(10, 5)
print(f"Area of rectangle: {area}")

for n in [4, 7, 12, 9]:
    result = "even" if is_even(n) else "odd"
    print(f"{n} is {result}")
📤 Expected Output
Hello Ravi! You are 15 years old. Hello Meera! You are 16 years old. Area of rectangle: 50 4 is even 7 is odd 12 is even 9 is odd
